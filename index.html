Welcome to GIraF!

  1. The first gif you see is retrieved from the giphy api using one of these random terms: cola, kebab, arrow, soccer, wasps, dog
  2. After that, we use ml5.js with the ImageClassifier to classify the gif.
  3. You can then choose one of the classifications to get another random gif.
  4. Repeat.
